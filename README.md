# X-FIT Pro Smartwatch Promotional Website

## Description
The X-FIT Pro Smartwatch Promotional Website is a responsive multi-page web project.

This project simulates a sports and wellness company launching a new fitness smartwatch product. The website promotes the smartwatch by highlighting its key features, pricing information, and product benefits, while also providing a contact and feedback form for potential customers.

The website was built using HTML5, CSS3, and Bootstrap 5 to ensure a modern, responsive, and user-friendly design. Interactive elements such as hover effects, floating animations, and smooth scrolling were implemented to enhance the overall user experience.

## Features
- Responsive multi-page layout (Home, Features, Contact)
- Navigation bar with active page links
- Hero section with product image and animation
- Product feature cards with icons
- Pricing section with call-to-action button
- Contact form integrated with Formspree
- Smooth scrolling and fade-in animations

## Technologies Used
- HTML5
- CSS3
- Bootstrap 5
- Bootstrap Icons
- Formspree (for contact form handling)

## How to Run the Project
1. Download or clone the repository.
2. Open `index.html` in a web browser.
3. Navigate through the website using the navigation bar.
